단어검색

* 온라인 쿠폰 : 사용자가 홈페이지에서 직접 다운로드(발급) 후 사용
: 다운로드 메뉴위치 - 로그인 후 > 마이페이지 > 쿠폰정보 > 발급가능 쿠폰
* 오프라인 쿠폰 : 먼저 쿠폰번호를 일괄 생성 후, 쿠폰번호 별로 사용자를 매칭시킨 후 사용하도록 한 형태
i) 발급가능대상을 미지정으로 선택한경우
: 목록에서 [쿠폰생성] 클릭 → 입력한 갯수만큼 쿠폰번호 생성됨 → 필요시마다 관리자가 쿠폰번호별로 사용자 ID 를 할당 (발급) → 결제시 바로 사용가능
ii) 그외 (특정 재직형태, 지정된 회원만 발급 등)
: 목록에서 [쿠폰생성] 클릭 → 조건에 맞는 시용자들에게 쿠폰번호 할당 (사용자에게 쿠폰번호 별도 안내필요) →
사용자는 홈페이지에서 안내받은 쿠폰번호를 등록 (발급) → 결제시 사용
: 쿠폰등록 메뉴위치 - 로그인 후 > 마이페이지 > 쿠폰정보
